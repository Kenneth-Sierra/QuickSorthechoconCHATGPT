# Implementación QuickSort en Java

## Descripción
Se implementó el algoritmo QuickSort utilizando la estrategia Divide y Vencerás para ordenar arreglos de enteros.

## Funcionamiento
El algoritmo selecciona un pivote y reorganiza el arreglo colocando los elementos menores a la izquierda y los mayores a la derecha. Posteriormente aplica el mismo procedimiento de forma recursiva.

## Escenarios evaluados
* Arreglo aleatorio
* Arreglo creciente
* Arreglo decreciente

## Tamaños utilizados
100, 1000, 5000, 10000 y 20000 elementos.

## Medición
Se utilizó el comando `System.nanoTime()` para medir el tiempo de ejecución.

## Uso de ChatGPT
Se utilizó ChatGPT como apoyo para:

* Comprensión del algoritmo QuickSort
* Implementación en Java
* Generación de pruebas experimentales
